About SoftwareCertHub

SoftwareCertHub is the GÉANT platform for managing software licensing certificates. It serves as the central tool for issuing, verifying, displaying, and maintaining the certificate lifecycle. Certificates are created and managed by the GÉANT licensing team, with automatic visibility in the GÉANT Software Catalogue (coming soon).


The platform also provides dynamically generated compact and large badges that can be embedded in software repositories, documentation (such as README files), and web pages to ensure transparent verification.


SoftwareCertHub is open source under the Apache-2.0 licence, with ongoing development focused on improving usability, interoperability, and integration.
